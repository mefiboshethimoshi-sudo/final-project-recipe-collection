# 🍽️ Recipe Collection & Cooking Website

## Overview
This is a fully functional **Recipe Collection & Cooking Website** designed to help users **browse recipes**, **search and filter by category**, **view detailed recipe information**, and **save favorites**. The project demonstrates advanced web development skills including **HTML5, CSS3, and JavaScript**, with features like **dark mode**, **interactive modals**, **localStorage persistence**, and **WhatsApp integration**.

This project is developed as part of a web development course to showcase **full-stack front-end skills** with modern, responsive, and attractive design.

---

## 🌟 Features

### Core Features
1. **Recipe Browsing**
   - Browse **10 preloaded recipes** with images and categories.
   - Recipes are displayed in an attractive card layout.

2. **Recipe Details & Modal**
   - Click on a recipe to open a **modal** showing **ingredients**.
   - Option to **add recipe to favorites** from the modal.

3. **Favorites**
   - Add or remove recipes from **favorites**.
   - Favorites are **persisted in localStorage**.
   - Favorites page displays images, titles, and remove button.

4. **Search & Autocomplete**
   - Search recipes by name with **autocomplete suggestions**.
   - Works instantly as you type.

5. **Dark Mode Toggle**
   - Switch between **dark mode and light mode** using button or `d` keyboard shortcut.

6. **Responsive Design**
   - Fully responsive using **CSS Flexbox and Grid**.
   - Works on **mobile, tablet, and desktop** devices.

7. **Contact Us**
   - Stylish **Contact form** with **Name, Email, Message** fields.
   - Send message **directly via WhatsApp**.

### Bonus Features
- **Animations & Micro-interactions** (hover effects, button transitions)
- **Random Cooking Tip** fetched via external API
- **Keyboard shortcuts** (Dark mode toggle via `d` key)
- **Data persistence** with `localStorage` for favorites
- **Search autocomplete** functionality
